# Orpheus Theme: Vinyl

A warm, analog-inspired theme with a spinning vinyl record for [Orpheus](https://github.com/collectif-pixel/orpheus).

## Preview

- Spinning vinyl record with grooves
- Album art displayed as the record label center
- Record spins when playing, stops when paused
- Warm sepia color palette with serif typography
- Smooth scale-in animation

## Installation

```bash
orpheus add @littlemorphee/orpheus-theme-vinyl
orpheus use @littlemorphee/orpheus-theme-vinyl
```
